## 🥔 开发

基于[chrome-extension-boilerplate-react-vite](https://github.com/Jonghakseo/chrome-extension-boilerplate-react-vite)进行开发
- 安装依赖 `pnpm i`
- 运行项目 `pnpm run dev`
- 安装拓展
  - 在浏览器中打开- chrome://extensions
  - 检查- Developer mode
  - 查找并单击- Load unpacked extension
  - 选择- dist 文件夹
- 打包 `pnpm run build`
